# Markdown Todo Standard (MDTS)

An open standard for markdown-based todo lists, empowering users with simplicity and plain-text portability.

## Vision

The Markdown Todo Standard aims to define extensible schemas for task management in pure Markdown. Whether tracking a simple grocery list or managing complex projects with paragraphs and attachments, MDTS provides a uniform language for tools to parse and display seamlessly.

## Core Pillars

1. **The Standard Schema**: Robust specifications defining everything from simple checked/unchecked lists `[ ]`, to projects with nested paragraphs and local/remote attachments.
2. **The Terminal UI (TUI)**: Fast, keyboard-centric interface for the terminal.
3. **The Graphical UI (GUI)**: Cross-platform mobile and desktop applications.
4. **The Web Hub**: A centralized landing page and distribution hub for apps supporting the standard.

## Architecture

This repository holds the open-source specification and serves as the hub for our official implementations.

- `/schema`: The official Markdown Todo Standard definitions.
- `/tui`: The Terminal User Interface reference implementation.
- `/gui`: The multi-platform GUI implementation (Mobile & Desktop).
- `/web`: The landing/distribution page.
