# 3. Projects and Complex Tasks

Tasks can act as containers for sub-tasks by nesting lists. The nested content must be indented deeper than the parent list marker.

## Nested Sub-tasks
These form projects.

- [ ] Prepare for Launch
  - [x] Create marketing assets
  - [/] Deploy landing page
    Waiting on DNS propagation.
  - [ ] Send newsletter

- [ ] Hire New Engineer
  - [x] Write job description
  - [x] Post on job boards
  - [ ] Interview candidates
    - [x] Screen resumes
    - [ ] Schedule technical interviews
  - [ ] Make offer

## Mixed Content Projects
You can interweave descriptions, metadata, and sub-tasks into a single block.

- [ ] Plan Company Retreat
  The retreat will be hosted in the mountains this year.
  
  budget: $5000
  organizer: @aricma
  
  - [x] Book cabins
  - [ ] Arrange transportation
  - [ ] Create itinerary
